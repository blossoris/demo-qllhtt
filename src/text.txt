
Website Quản Lý Lớp Học Trực Tuyến

Chức năng chính
1/ Cho sinh viên
  Đăng ký / Đăng nhập (Auth)
  Xem danh sách môn học
  Xem tài liệu, bài giảng

2/ Cho giảng viên
  Đăng nhập
  Môn học ==> C-U-D
  Tìm kiếm và filter môn học
  Upload tài liệu (PDF, link, video) ==> C-U-D

3/ Cấu trúc kỹ thuật
  React Router
  Ant design, TailwindCSS hoặc shadcn/ui (UI đẹp, nhanh)
  Axios để gọi API
  useState, useEffect, useContext
  React Hook Form (quản lý form)
  Component hóa giao diện: Header, Sidebar, Card, Form...

4/ Backend
	Golang
		
5/ Kết quả đầu ra
	Source code GitHub
	Demo video
	
6/ Các bạn làm Web và App  tạo dữ liệu bằng file json để làm
Tìm hiểu ReactJS nâng cao
Bắt đầu xử lý dữ liệu, chia module, gọi API, quản lý trạng thái.
- Gọi API với fetch hoặc axios
- Form handling, validation (React Hook Form)
- useContext (context API), prop drilling
- React Router (v6+): routing, nested routes
- Xử lý lỗi, loading, conditional rendering
- Component cấu trúc lại theo page/module
